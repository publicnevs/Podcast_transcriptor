PodScribe ist ein privates Self-Hosting-Projekt. Die App wurde entwickelt, um Podcast-Transkripte automatisiert zu erstellen, durchsuchbar zu machen und mit KI-Tools weiterzuverarbeiten. Transkription via Google Gemini API · Audio-Download via yt-dlp · Deployment via Docker auf Synology NAS.
⚡ Feature-Übersicht
📻

Podcast-Management

✓
RSS-Feed Abonnement
Podcast abonnieren via RSS-URL, YouTube-Kanal oder direkter Audio-URL
✓
OPML-Import
Alle Podcasts auf einmal aus Apple Podcasts, Pocket Casts oder anderen Apps importieren
✓
Auto-Transkription
Neue Folgen werden automatisch erkannt und transkribiert (konfigurierbar pro Podcast)
✓
Flexible Abo-Optionen
Max. Folgen limitieren, Intervall konfigurieren, nur bestimmte Folgen transkribieren
🎙️

Transkription

✓
Gemini 2.5 Flash (Cloud)
Schnelle, günstige Cloud-Transkription. Rate Limits gelten pro Minute. Empfohlen.
✓
Lokales Whisper-Backend
faster-whisper (Modelle: tiny/base/small) — offline, kostenlos, aber langsamer auf schwacher CPU
✓
Sprecher-Erkennung
Host/Gast werden automatisch erkannt und farblich im Transkript markiert
✓
Zeitstempel
Jeder Absatz hat einen klickbaren Zeitstempel — springt direkt zur Stelle im Audio-Player
🤖

KI-Features

✓
Auto-Zusammenfassung
5-Satz-Zusammenfassung jeder Folge, automatisch nach der Transkription
✓
Key Takeaways
6–10 Bullet Points der wichtigsten Erkenntnisse, direkt auf der Episodenseite
✓
Automatische Kapitel
Themenblöcke mit Zeitstempeln werden erkannt — klickbare Navigation im Transkript
✓
Deutsche Übersetzung
Englische (oder andere) Transkripte per Klick ins Deutsche übersetzen
✓
Für KI kopieren
Transkript + Metadaten optimal formatiert für Claude / Gemini / ChatGPT
🎧

Audio-Player mit synchronem Transkript

✓
Synchrones Mitlesen
Der aktuelle Absatz wird hervorgehoben und scrollt automatisch mit dem Audio mit
✓
Klick-to-seek
Klick auf einen beliebigen Transkript-Absatz → Audio springt direkt zur Stelle
✓
Geschwindigkeitssteuerung
1× · 1.25× · 1.5× · 1.75× · 2× — ideal für effizientes Nachhören
✓
15-Sekunden-Sprung
Vor/Zurück-Sprung per Knopf, Range-Support für Seeking
🔍

Bibliothek & Suche

✓
Volltextsuche
SQLite FTS5 — blitzschnelle Suche über alle Transkripte, auch sehr große Bestände
✓
Ungelesen-Tracking
Badge pro Podcast, Scroll-Position wird gespeichert, "als gelesen" markieren
✓
Leseliste
Folgen für später vormerken, persönliche Notizen pro Episode
✓
Filter & Sortierung
Nach Podcast, Zeitraum, Lesestatus und Transkriptions-Status filtern
📦

Export

✓
TXT / Markdown / PDF
Einzelne Episode in verschiedenen Formaten exportieren und herunterladen
✓
Bulk-Export
Alle Folgen eines Podcasts als eine große Markdown-Datei — ideal für AI-Research
✓
Drucken/PDF
Druckansicht direkt aus dem Browser — sauber formatiert ohne UI-Chrome
📰

PodScribe Zeitung (Digest)

✓
Journalistische Artikel
Gemini 2.5 Pro generiert ausführliche, echte Redaktionstexte (1200–2000 Wörter) aus Transkripten
✓
Wochenzeitung
Alle Folgen der letzten 7 Tage zu einem kompakten Wochenbericht zusammengefasst
✓
Themen-Report
Manuell ausgewählte Folgen zu einem thematischen Artikel kombinieren
✓
Podcast-Portrait
Alle Folgen eines Podcasts analysieren — Stil, Qualität, wiederkehrende Themen
📱

PWA & Mobile

✓
Progressive Web App
Als echte App auf dem Homescreen installierbar (Android/iOS) — kein App Store nötig
✓
Offline-Lesen
Service Worker cached Transkripte — auch ohne Internet lesbar (z.B. im Zug)
✓
Mobile-first Design
Bottom-Navigation, große Touch-Targets, Schriftgrößen-Regler, Dark Mode
✓
Push-Benachrichtigungen
Über ntfy.sh auf Android-Handy — Benachrichtigung wenn Transkript fertig ist
